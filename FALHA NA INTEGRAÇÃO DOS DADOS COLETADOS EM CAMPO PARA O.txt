FALHA NA INTEGRAÇÃO DOS DADOS COLETADOS EM CAMPO PARA O PROCESSO DE SECAGEM 
    MÉTODO 
      PROCESSO DE COLETA DE DADOS EM CAMPO MAL DEFINIDOS OU INCONCISTENTES 
      FALHA  DE  PADRONIZAÇÃO NA ENTRADA DE DADOS (TALHÃO , TIPO DE GRÃO, QUANTIDADE, PESO AMOSTRA, 
      TEMPERATURAS). 
      PROCEDIMENTOS DE INTEGRAÇÃO DE DADOS MANUAIS OU PROPENSOS A ERRO 
      METODOLOGIA DE SECAGEM NÃO CONSIDERADA NA COLETA INICIAL. 
      FALTA DE VALIDAÇÃO DOS DADOS NO MOMENTO DA COLETA. 
    MEIO AMBIENTE 
      CONDIÇÕES CLIMÁTICAS ADVERSAS EM CAMPO QUE DIFICULTAM A COLETA PRECISA (CHUVA, POEIRA). 
      INTERFERÊNCIA NA CONECTIVIDADE DE REDE EM CAMPO PARA A TRANSMISSÃO DE DADOS. 
      INFRAESTRUTURA DE TI INSTÁVEL OU INSUFICIENTE PARA A INTEGRAÇÃO 
      AMBIENTE  DE  DESENVOLVIMENTO  /  PRODUÇÃO  NÃO OTIMIZADO PARA A INTEGRAÇÃO DE DADOS EM LARGA 
      ESCALA. 
    MEDIDA 
      SENSORES  OU  EQUIPAMENTO  DE  COLETA  DE  DADOS  (TEMPERATURA  DO  GRÃO  /  AMBIENTE,  PESO) 
      DESCALIBRADOS OU IMPRECISOS. 
      UNIDADES  DE  MEDIDA  INCONSISTENTES  ENTRE OS DADOS COLETADOS E OS REQUISITOS DO PROCESSO DE 
      SECAGEM. 
      ERROS DE TRANSIÇÃO OU REGISTRO MANUAL DAS MEDIÇÕES. 
      FREQUÊNCIA DE COLETA DADOS INSUFICIENTE PARA MONITORAMENTO DA SECAGEM. 
    MAQUINA 
      HARDWARE DE COLETA DADOS EM CAMPO (COLETORES,TABLETS) COM FALHAS OU SOFTWARE DESATUALIZADO. 
      SISTEMA DE SOFTWARE PARA A COLETA DADOS EM CAMPO COM BUGS OU LIMITAÇÕE 
      SISTEMA DE INTEGRAÇÃO DE DADOS (MIDDLEWARE, APIS) COM PROBLEMAS DE PERFORMACE OU FALHAS. 
      SERVIDORES OU BANCO DE DADOS ONDE OS DADOS SÃO ARMAZENADOS E PROCESSADOS COM GARGALOS. 
    MATERIAL 
      QUALIDADE DOS DADOS COLETADOS ( DADOS INCOMPLETOS, INCONSISTENTES, DUPLICADOS. 
      DADOS DO PROCESSO DE COLHEITA NÃO COMPATÍVEIS COM OS REQUISITOS COM PROCESSO DE SECAGEM. 
      METADADOS AUSENTES OU INCORRETO QUE DIFICULTAM A INTERPRETAÇÃO DOS DADOS. 
      DEPENDÊNCIA DE MÚLTIPLAS FONTES DE DADOS QUE NÃO SE COMUNICAM BEM. 
    MÃO DE OBRA 
      TREINAMENTO INADEQUADO DA EQUIPE DE CAMPO PARA A COLETA DE DADOS. 
      FALTA DE CONCIENTIZAÇÃO SOBRE A IMPORTÂNCIA DA PRECISÃO DOS DADOS. 
      RESISTENCIA À ADOÇÃO DE NOVAS FERRAMENTAS OU PROCESSOS DE COLETAS. 
      FALTA DE COMUNICAÇÃO ENTRE A EQUIPE DE CAMPO, DESENVOLVIMENTO E PROCESSO DE SECAGEM. 
      ROTATIVIDADE DA EQUIPE RESULTADO EM PERDA DE CONHECIMENTO SOBRE OS PROCEDIMENTOS.
